Pipes
=
Connect all pipes to win

Install
-
1. Download the latest release
2. Unpack the zip file
3. Run the exe file and play

Controls
-
- Left Click on a cell to rotate a pipe 90° clockwise
- Escape to exit to level selector

Level Editor
=
Make your own level of any size and shape

Controls
-
- Use the console to create a level or open an existing one
- Left Click on a cell to rotate a pipe 90° clockwise
- Right Click on a cell to change the shape of the pipe
- Press 'R' to randomly rotate all pipes
- Press 'S' and continue in console to save your level

Opening your levels in game
-
1. Your levels are saved in `/levelmaker/_internal/levels`
2. Move the `.txt` file to the main game's `/pipes/_internal/levels`
3. Start the game
4. Press 'X' and enter the level name in the console
